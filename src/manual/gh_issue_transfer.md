

## gh issue transfer

Transfer issue to another repository

```
gh issue transfer {<number> | <url>} <destination-repo>
```

<dl class="flags">
	<dt><code>-R</code>, <code>--repo &lt;[HOST/]OWNER/REPO&gt;</code></dt>
	<dd>Select another repository using the [HOST/]OWNER/REPO format</dd>
