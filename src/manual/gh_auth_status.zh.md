## gh auth status

```
gh auth status [flags]
```

验证并显示有关身份验证状态的信息。

此命令将测试 gh 知道的每个 GitHub 主机的身份验证状态，并报告任何问题。

### Options

<dl class="flags">
	<dt><code>-h</code>, <code>--hostname &lt;string&gt;</code></dt>
	<dd>检查的主机</dd>


<dt><code>-t</code>, <code>--show-token</code></dt>
<dd>展示 身份令牌</dd>


</dl>
