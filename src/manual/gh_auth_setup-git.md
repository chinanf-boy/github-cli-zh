

## gh auth setup-git

Configure git to use GitHub CLI as a credential helper

```
gh auth setup-git [flags]
```

### Options


<dl class="flags">
	<dt><code>-h</code>, <code>--hostname &lt;string&gt;</code></dt>
	<dd>The hostname to configure git for</dd>
</dl>
