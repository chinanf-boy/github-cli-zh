

## gh codespace code

Open a codespace in Visual Studio Code

```
gh codespace code [flags]
```

### Options


<dl class="flags">
	<dt><code>-c</code>, <code>--codespace &lt;string&gt;</code></dt>
	<dd>名字</dd>

	<dt><code>--insiders</code></dt>
	<dd>Use the insiders version of Visual Studio Code</dd>
</dl>
