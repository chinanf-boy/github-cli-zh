## gh codespace delete

删除代码空间

```
gh codespace delete [flags]
```

### Options

<dl class="flags">
	<dt><code>--all</code></dt>
	<dd>删除所有</dd>

<dt><code>-c</code>, <code>--codespace &lt;string&gt;</code></dt>
<dd>名称</dd>

<dt><code>--days &lt;N&gt;</code></dt>
<dd>删除过去 N 天</dd>

<dt><code>-f</code>, <code>--force</code></dt>
<dd>强制删除</dd>

<dt><code>-r</code>, <code>--repo &lt;repository&gt;</code></dt>
<dd>删除一个库</dd>

</dl>
