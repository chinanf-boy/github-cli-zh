## gh codespace logs

访问代码空间日志

```
gh codespace logs [flags]
```

### Options

<dl class="flags">
	<dt><code>-c</code>, <code>--codespace &lt;string&gt;</code></dt>
	<dd>名字</dd>

<dt><code>-f</code>, <code>--follow</code></dt>
<dd>日志跟随滚动</dd>

</dl>
