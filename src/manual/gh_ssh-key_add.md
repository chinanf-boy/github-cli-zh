## gh ssh-key add

Add an SSH key to your GitHub account

```
gh ssh-key add [<key-file>] [flags]
```

### Options


<dl class="flags">
	<dt><code>-t</code>, <code>--title &lt;string&gt;</code></dt>
	<dd>Title for the new key</dd>
</dl>
