## gh gist edit

编辑一个 GIST

```
gh gist edit {<id> | <url>} [flags]
```

### Options

<dl class="flags">
	<dt><code>-a</code>, <code>--add &lt;string&gt;</code></dt>
	<dd>新增一个文件</dd>

<dt><code>-f</code>, <code>--filename &lt;string&gt;</code></dt>
<dd>选择一个文件编辑</dd>

</dl>
