## gh codespace code

在 Visual Studio Code 中，打开代码空间

```
gh codespace code [flags]
```

### Options

<dl class="flags">
	<dt><code>-c</code>, <code>--codespace &lt;string&gt;</code></dt>
	<dd>名字</dd>

<dt><code>--insiders</code></dt>
<dd>Visual Studio Code 内部版本</dd>

</dl>
