## gh extension upgrade

Upgrade installed extensions

```
gh extension upgrade {<name> | --all} [flags]
```

### Options


<dl class="flags">
	<dt><code>--all</code></dt>
	<dd>Upgrade all extensions</dd>

	<dt><code>--force</code></dt>
	<dd>Force upgrade extension</dd>
</dl>
