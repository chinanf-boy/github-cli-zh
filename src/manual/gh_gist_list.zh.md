

## gh gist list

列出你的注册医生

```
gh gist list [flags]
```

### Options

<dl class="flags">
	<dt><code>-L</code>, <code>--limit &lt;int&gt;</code></dt>
	<dd>Maximum number of gists to fetch</dd>

```
<dt><code>--public</code></dt>
<dd>Show only public gists</dd>

<dt><code>--secret</code></dt>
<dd>Show only secret gists</dd>
```

</dl>
