## gh gist list

列出你的 gist

```
gh gist list [flags]
```

### Options

<dl class="flags">
	<dt><code>-L</code>, <code>--limit &lt;int&gt;</code></dt>
	<dd>列出的数量</dd>

<dt><code>--public</code></dt>
<dd>只给出 public gists</dd>

<dt><code>--secret</code></dt>
<dd>只给出 secret gists</dd>

</dl>
