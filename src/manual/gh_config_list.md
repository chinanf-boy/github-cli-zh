## gh config list

Print a list of configuration keys and values

```
gh config list [flags]
```

### Options


<dl class="flags">
	<dt><code>-h</code>, <code>--host &lt;string&gt;</code></dt>
	<dd>Get per-host configuration</dd>
</dl>
