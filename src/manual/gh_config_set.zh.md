

## gh config set

使用给定密钥的值更新配置

```
gh config set <key> <value> [flags]
```

### Options

<dl class="flags">
	<dt><code>-h</code>, <code>--host &lt;string&gt;</code></dt>
	<dd>Set per-host setting</dd>
</dl>

### Examples

```bash
$gh配置集编辑器vim$gh配置集编辑器“code--wait”$gh配置集git\_协议ssh--host github。com$gh配置集提示已禁用
```
