## gh workflow enable

```
gh workflow enable [<workflow-id> | <workflow-name>]
```

启用工作流，允许它在列出工作流时，运行和显示。

	<dt><code>-R</code>, <code>--repo &lt;[HOST/]OWNER/REPO&gt;</code></dt>
